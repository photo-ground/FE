## ✏️ 서비스 소개

> “아마추어 사진작가”와 “행복한 대학생”을 연결해주는 **스냅 촬영 전문 통합 관리 서비스**

<br/>

## ✨ 주요 기능

### ⚙️ 교내 포토스팟 맵 제공

> 포토스팟 탭에서 페이지에서 학교 별 포토스팟을 확인할 수 있습니다.

### ⚙️ 사진 작가 프로필 및 포트폴리오 제공

> 사진작가 탭에서 작가들의 정보와 작품을 확인할 수 있습니다.

### ⚙️ 진행 단계별 실시간 소통

> 로그인 후 예약 탭에서 작가와의 예약을 실시간으로 확인할 수 있습니다.
